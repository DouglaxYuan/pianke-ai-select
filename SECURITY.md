# Security Policy

## Sensitive Data

This project processes local photos. Do not commit:

- private photos
- client photos
- generated review sessions
- local model caches
- logs containing file paths
- API keys used by online or LLM-assisted modes

## Reporting Issues

For upstream Pianke security or licensing issues, refer to the upstream project:

https://github.com/zhaoyue4810/pianke

For issues specific to this fork, open a private issue or contact the repository maintainer before sharing private photo paths or logs.
