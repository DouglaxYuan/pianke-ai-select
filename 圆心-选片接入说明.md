## 和我们之前思路的对比

片刻已经覆盖我们第一阶段选片系统里的很多底层能力：

- 技术废片筛选：模糊、曝光、质量评分。
- 人像筛选：人脸检测、人脸清晰度、闭眼倾向、脸部 embedding。
- 相似图分组：DINOv2 语义特征 + 人脸聚类，把同一瞬间归组。
- 人工最终决策：左右擂台 PK，符合摄影师最终审美由人把关的流程。
- RAW/JPG 工作流：支持 RAW+JPG 同名双拍，归档时伴随文件一起处理。

片刻暂时缺少我们项目里更个性化的部分：

- 不知道圆心自己的后期偏好。
- 不知道午饭饭方向的色彩、光比、氛围倾向。
- 不会判断一张图是否“适合后期修成目标风格”。
- 不会把 winners 自动接入 Lightroom / PixCake / ComfyUI。

## 推荐流程

1. 用片刻专家模式对一次拍摄做第一轮筛选。
2. 先使用复制模式，不要一开始移动原片。
3. 把 `winners/` 和需要复核的候选图送入我们自己的偏好评分器。
4. 用午饭饭参考图和圆心成片偏好做二次排序：
   - 人脸清晰、睁眼、可修性；
   - 侧逆光、轮廓光、空气感；
   - 蓝青环境与暖肤色对比；
   - 暗部不死黑，肤色干净通透；
   - 是否适合后续 PixCake / Lightroom / ComfyUI 修图。
5. 高分图进入后期流程，低分但有潜力的图进入候选池。

`/Volumes/英睿达 P3/AI选片/pianke/停止-片刻专家版.command`

或命令行：

```bash
cd "/Volumes/英睿达 P3/AI选片/pianke"
source ".venv/bin/activate"
python app.py --port 5057
```

打开：

`http://127.0.0.1:5057`

## 注意

- 首次真实跑照片时建议选“专家模式 + 复制模式”。
- 片刻的移动模式会移动原片到 winners/losers，确认流程前先别用。
- 土豪模式会把缩略图发给大模型服务商；隐私照片优先用极速/专家。
- 如果外置盘休眠，先运行桌面的挂载脚本或重新插拔硬盘。
